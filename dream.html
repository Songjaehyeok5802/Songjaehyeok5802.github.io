Dream
Dream
  • Tools /
    Unity / 3DMax / C#
  • Date /
    2017

20대의 입장에서 동심의 세상을 만들었던 프로젝트 입니다.
그래픽이 화려한 요즘의 게임들과 반대되는 프로젝트를 만들고자 low poly 형식으로 만들었습니다.
동심의 주를 이루는 큰 성이 있는 섬을 중간에 배치하고 주변에는 어렸을 때 상상 했었던 여러 세상들을
주변 섬으로 배치하였습니다. 1인칭과 3인칭 모두 체험할 수 있게 만들어 사용자의 자율성을 높였습니다.

인트로 화면입니다.
1인칭과 3인칭 플레이를 선택할 수 있는 화면입니다.
Exit 버튼을 클릭 시 유니티 게임이 나가집니다.

1인칭 플레이 화면입니다.
키보드로 컨트롤 가능하여 각 섬들을 오갈 수 있습니다.
사용자가 원하는 구역들을 직접 가서 볼 수 있는 장점이 있고
섬 밖으로 떨어질 경우 인트로 화면으로 나가집니다.

3인칭 플레이 화면입니다.
각 버튼을 클릭 시 카메라가 정해진 좌표로 움직입니다.
섬을 다양한 시점에서 볼 수 있으며 각 오브젝트들의
애니메이션들을 더욱 실감 나게 감상할 수 있습니다.
